# Installation

This document describes how to install **IntelliJWOPlugin** starting with **v0.7.0**.

## Install via JetBrains Marketplace (v0.7.0+)

1. Open **Settings / Preferences → Plugins**.
2. Go to the **Marketplace** tab.
3. Search for the plugin and click **Install**.
4. Restart the IDE when prompted.

## Install from GitHub Releases (v0.7.0+)

1. Download the plugin archive (`.zip`) from the GitHub Releases page for the desired version.
2. Open **Settings / Preferences → Plugins**.
3. Click the **gear icon** and choose **Install Plugin from Disk…**
4. Select the downloaded archive and restart the IDE.

## Upgrading from manual installations before v0.7.0

If you installed this plugin **manually** (Install from Disk) with a version **before v0.7.0**, you must:

1. **Uninstall** the old plugin in **Settings / Preferences → Plugins**.
2. **Restart** the IDE.
3. Install the new version (either from the Marketplace or from GitHub Releases).

This is required to avoid the IDE keeping an outdated plugin artifact side-by-side with the newer plugin version.
